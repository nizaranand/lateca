This plugin adds a widget which let you display a list of categories in your sidebar with more features and flexibility.

== Description ==

This is a simple, light weight and little widget that adds in some useful amazing functionality that WordPress doesn't offer in its category widget. Some Features of this Widget are:-

* Manage Which Category to Show and Which to Hide.
* Enable Disable Post Count.
* Hide Or Show Empty Categories with no posts.
* Limit Max Number of Categories to be Displayed...etc.

== Installation ==

= >>>First Way =

1. Choose :Add New: From Plugins Drop Down Menu in WordPress Admin Area
2. Enter 'Easy Categories Management' in search field and hit enter
3. Below plugin name Click 'Install Now'
4. After Installation Complete Choose Activate
5. Position the widget in the desired sidebar through the 'widgets' admin page

= >>>Second Way =

1. Choose 'Add New' From Plugins Drop Down Menu in WordPress Admin Area
2. Click 'Upload' on the sub menu
3. Select The .zip file you have downloaded
4. After Installation Complete Choose Activate
5. Position the widget in the desired sidebar through the 'widgets' admin page.

= >>>Third Way =

1. Upload the folder ::Easy-Categories-Management:: to the `/wp-content/plugins/` directory using any FTP or Shell
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Position the widget in the desired sidebar through the 'widgets' admin page
